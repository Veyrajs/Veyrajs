`SelectionController` adds an interactive selection + transform layer over any stage: click to
select, drag to move, and use the handles to resize and rotate. Hand it a `History` and every
change becomes an undoable command — no extra wiring.

## The transformer

Add shapes, select one, and drag its handles. Undo/redo is wired to every transform:
